A showcase of my data engineering work

🔥

ETL Pipeline in Apache Spark

Developed a PySpark ETL pipeline processing millions of records with data quality validation and Parquet storage. Deployed PostgreSQL via Docker and built analytical dashboards for reporting.

PySpark PostgreSQL Docker Parquet
🕸️

Large-Scale Web Crawler Data Processing

Scraped 100,000+ web listings and built an automated data cleaning and quality assurance pipeline. Benchmarked multiple processing libraries to optimize performance and throughput.

Python Web Scraping Pandas Benchmarking

Azure End-to-End Data Engineering Project

Built an automated data pipeline from on-premises SQL Server to Azure using Medallion Architecture (Bronze, Silver, Gold). Modeled data into star schema and developed interactive Power BI dashboard for business reporting.

Azure Medallion Architecture Power BI Star Schema

Vehicle Emissions & Fuel Efficiency Analysis
